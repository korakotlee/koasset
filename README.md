# KoAsset - Consolidated Estate Report

KoAsset helps you manage and visualize your estate, assets, and beneficiaries.

## Try it out
You can try the live version here: [https://koasset.vercel.app/](https://koasset.vercel.app/)

### Instructions
1. **Add Data**: Enter your assets and beneficiaries to see the visualization.
2. **Export/Import**: Backup your data to a JSON file or restore it.
3. **Print Report**: Generate a professional consolidated estate report ready for printing.

**Privacy First**: All data is encrypted with a 4-digit PIN using AES-256-GCM and stored in your browser's local storage. Your financial information never leaves your device.

**⚠️ IMPORTANT SECURITY NOTE**:
This application has **NO recovery mechanism**. If you forget your 4-digit PIN, all your stored data and backups are **permanently and irreversibly lost**. We do not store your PIN or any master keys.

## Development

This project is built with React, TypeScript, and Vite.

### Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
